# Voodoo-SAP

Example code for integrating pick to light devices from Voodoo Robotics with SAP.

## Description

This project provides integration examples and utilities for connecting Voodoo Robotics pick-to-light devices with SAP systems. It enables seamless communication between warehouse automation hardware and enterprise resource planning software.

## Features

- Integration with Voodoo Robotics pick-to-light devices
- SAP system connectivity
- Real-time data synchronization
- Error handling and logging
- Configurable mapping between devices and SAP processes
